# Amazon Sales Analytics Dashboard (Power BI)

# Project Overview
This project analyzes Amazon sales data to uncover insights on revenue, profitability, customer behavior, and product performance using Power BI.
I Analyzed Amazon sales data using Power BI to uncover insights on revenue, profitability, and customer behavior. Built interactive dashboards with drill-through and advanced analytics to evaluate trends, regional performance, product categories, and the impact of discounts on sales and profit.

The dashboard enables data-driven decision-making through interactive visuals, drill-through functionality, and advanced analytics.

#Objectives
- Analyze sales performance across product categories and regions
- Evaluate profitability and profit margins
- Understand the impact of discounts on revenue
- Identify trends and customer behavior patterns

# Tools & Technologies
- Microsoft Power BI
- DAX (Data Analysis Expressions)
- Power Query
- CSV Dataset

# Dashboard Features

# 1. Overview Dashboard
- Total Revenue, Profit, Orders, and Ratings (KPIs)
- Sales by Product Category
- Sales by Region
- Revenue Trends Over Time
- Payment Method Breakdown

# 2. Drill-Through Analysis
- Detailed insights by product category or region
- Interactive filtering and navigation
- Transaction-level data view

# 3. Advanced Analysis
- Profit Margin Analysis
- Discount vs Revenue Insights
- Customer Behavior Analysis
- Regional Profitability

# Key Insights
- Some categories generate high revenue but lower profit margins
- Discounts increase sales volume but may reduce profitability
- Certain regions outperform others in both revenue and profit
- Higher-rated products tend to drive more sales
